# web_booking_bengkel
Web Booking Servis Mobil adalah aplikasi berbasis web yang digunakan untuk melakukan pemesanan servis mobil secara online.

Sistem ini memiliki dua aktor utama:

- Pelanggan (User)
- Admin

Pelanggan dapat melakukan booking servis dan mendapatkan kode akses untuk mengedit data, sedangkan admin dapat mengelola seluruh data booking melalui dashboard.

Project ini dibuat menggunakan:

- PHP Native
- MySQL
- HTML
- CSS
- JavaScript

Tanpa framework.
